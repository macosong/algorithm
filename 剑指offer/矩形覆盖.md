我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

#### Solution

```java
public class Solution {
    public int RectCover(int target) {
        if(target <= 2)    return target;
        int a = 1;
        int b = 2;
        int c = a + b;
        int count = 3;
        while(count++ != target){
            a = b;
            b = c;
            c = a + b;
        }
        return c;
    }
}
```
